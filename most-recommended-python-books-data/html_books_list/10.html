5 Excellent Books to Learn Python in 2020

Julie Perilla Garcia
Follow
Jan 23 · 5 min read

Online learning is easier than ever. There are hundreds of ways to learn programming languages online, especially if you want to learn Python.

But I still love to learn from books. When I open a programming book, I often disconnect from the internet and code in a bubble without distraction.

With books, I can get into a flow of deep thinking that helps me to really understand the material.

Here are five books that I recommend to take your Python skills from beginner to advanced in just a few months.

Python Crash Course

Python Crash Course, by Eric Matthes, is a fantastic book for absolute beginners.

This book gives you a solid foundation in the basics of Python alongside several practical code examples for real-world applications that you can start using right away.

It even takes you step-by-step through creating a simple web application with Django that includes authentication. This chapter guides you through deploying your application to Heroku once built, which is nice. Often with tutorials, you are left wondering how you would take your application to production, but this includes that step.

This book is a delight to read and helps you to piece together how you can use Python in different aspects of a real application.

Effective Python

I’ve been reading “Effective Programming” books for years. I started with Effective C++ in the early 2000s, and it took the programming that I had learned in college to the next level.

Since then, I have read Effective C#, More Effective C#, and Effective Python. All of these books have a similar theme. Once you know the basics of a programming language, you should learn the best practices to learn how to write clean, readable, reusable code in that language. The “Effective Programming” books do a fantastic job of helping you on that journey from beginner to advanced programmer.

Effective Python, by Brett Slatkin, is no exception. This book is one that you can either read from cover to cover or keep on your desk and reference it from time to time to see if anything applies to what you are currently working on.

For example, once you have a few scripts under your belt, with several functions, you can go through Chapter 5, Functions, in this book to take your function writing skills up a level.

I love this pattern in my own learning. First, figure out how to do something, then once you have the basics down, hone in on what needs improving. This is how you master a programming language.

Python 3 Object-Oriented Programming

If you are writing a full-stack web application or have a larger data science project, you will most likely want to start creating reusable components.

While it’s not necessary to write Python in an Object-Oriented manner, it is a pattern that nicely encapsulates your code, or creates the illusion of encapsulation, and makes it more readable for other developers.

Not to mention, it makes your applications more easily testable and maintainable.

Python 3 Object-Oriented Programming, by Dusty Phillips, goes in-depth into how to start creating classes with Python and the best practices surrounding them. It also is great for learning some common design patterns used in Python. Some of them are implemented differently in other strongly-typed languages, so if you are coming from a C++, C#, or Java background, it’s still worth a read to see how these patterns are implemented in Python. It may not always be the way that you think.

Python Tricks Book

When you’re ready to take your Python programming to the next level, the Python Tricks Book, written by Dan Bader, is an excellent choice.

I have been programming with Python for years, but read this book recently and learned a lot. It takes common constructs in Python and uses them in ways you may not think of. This book is excellent for beginners to experts alike.

You can read this book in any order. Each section is self-contained, so browse through the contents and find something that interests you.

I love this book!

Fluent Python

I have to admit, I have just started Fluent Python, by Luciano Ramalho, because so many others recommended it as an expert level Python book.

This book describes best practices for advanced features of Python like generators and when to use them, various ways to implement concurrency, and an entire chapter on meta-programming.

I will do a more thorough review of this when I finish this book, but it got great reviews for advanced Python programmers, so I have high hopes.

With each new book you read or course you take, you will begin to see gaps in your knowledge filling in and your mind making connections between things it hasn’t before.

If you want to be an expert programmer, it’s crucial to learn as many different constructs and patterns that you can. You may not be able to implement them correctly from memory and that is fine. However, just having the knowledge that certain programming methods exist, will allow you to look them up when you need them, and broaden your ability to write great code.

Happy coding!
