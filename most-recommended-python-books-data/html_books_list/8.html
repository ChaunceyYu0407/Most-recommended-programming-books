8 Must-Read Books for Python Programmers

Books for Mastering Python in different domains

Jun Wu
Follow
Dec 28, 2019 · 4 min read

Python is one of the most popular programming languages. It’s versatility and simplicity makes it one of the most beloved programming languages for a lot of programmers. A lot of programmers like to go through courses to learn how to program. But, if you are like me, who likes to flip through a book at my pace while working on a project, then you need this list of 8 books for your Python adventures. These books will not only teach you the basics of the language but they will allow you see possibilities of applications, such as data science, machine learning, web development and more.

Disclaimer: There are no affiliate links in this post. This post is for information purposes only.

Beginner Python Books

Learn Python — Mark Lutz

This book is easy to follow and hands-on introduction to Python that any programmer can dive into. You learn all the basics while writing a decent amount of code. It’s probably the one book that can prepare you conceptually and in practice to delve into more advanced concepts.

Fluent Python — Luciano Ramalho

Learn to write simple and effective Python code from the beginning with this book. Luciano shows you all the ways to leverage the best parts of Python to write concise and readable code.

Intermediate Python Books

Python Cookbook -David Beasley, Brian K. Jones

Once you have some experience writing Python code, it’s time to dive into this book to learn all the ins and outs of more advanced concepts. If you are working on a project, this book is the good reference book.

Python Tricks: A Buffet of Awesome Python Features — by Dan Bader

This book is full of Python tricks that you want to learn from the beginning. It can save you a lot of time taking the wrong turn in your Python code. Dan Bader teaches you all the tricks in the Python standard library that makes programming in Python fun again.

Web Development

Flask Web Development — Miguel Ginsberg

Use Python for your web development. Using Python-based microframework, Flask, you will develop a complete, real-world application with this book. You will appreciate the hands-on nature of this book and how easy it is to develop a web application on your own with Python.

Two Scoops of Django 1.11 — Daniel Rooy Greenfeld, Audrey Roy Greenfield

This book will walk you through all the ins, outs, and best practices of using the Django web framework. There’s a lot of techniques, tips and tricks in this book that any web programmer can benefit from.

Data Science and Machine Learning

Python for Data Analysis — Wes McKinney

This is a good book for data engineers to learn how to manipulate, process, clean and crunch data in the latest version of Python. It teaches you the basics of how to use Python libraries such as pandas, numpy, etc.. for data science.

Introduction to Machine Learning with Python — by Andreas C. Müller and Sarah Guido

This book will take you through how to train models, select algorithms and tune parameters. If you want to become a machine learning engineer, then this book can get you started on that path. With examples, you learn about the process as well as the algorithms, and the code.
