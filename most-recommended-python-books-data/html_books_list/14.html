totalwise wrote: The books version of python is not compatible with the pi python. Apparantly a lot of the operations dont work.
There are two active versions of Python. Python2 and Python3, Check which one the book is written for. This might explain the differences.

You run run either version on a raspberry pi using either
python - For Python2
python3 - For Python3

The easiest difference to spot is with the print statement. Print is a function in python3.
Python 2 wrote:print "Hello World"
Python 3 wrote:print("Hello World")
